# Website Quản Lý Bán Điện Thoại

Công nghệ: ASP.NET Web Application | HTML/CSS/JS | RESTful API | SQL Server

  Giới thiệu
- Website quản lý bán điện thoại là hệ thống hỗ trợ cửa hàng trong việc quản lý sản phẩm, đơn hàng, khách hàng và tồn kho.
- Dự án được xây dựng theo mô hình 3 lớp và sử dụng ASP.NET kết hợp API để xử lý dữ liệu, mang lại kiến trúc rõ ràng – dễ bảo trì – dễ mở rộng.

  Mục tiêu
- Xây dựng hệ thống bán hàng trực tuyến chuyên nghiệp.
- Quản lý sản phẩm (điện thoại), tồn kho, đơn hàng, tài khoản.
- Cung cấp API cho giao diện frontend truy xuất dữ liệu.
- Hỗ trợ bán hàng – theo dõi doanh thu – quản lý tài khoản người dùng.
